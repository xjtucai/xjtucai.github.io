# A4. 如何运行.js文件？

要运行 JavaScript 文件（.js 文件），你可以使用 Node.js。Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时，它允许你在服务器端运行 JavaScript。

# 一、在命令行中运行 JavaScript 文件

  1. 安装 Node.js: 首先，确保你的计算机上已经安装了 Node.js。你可以从 Node.js 官方网站 下载并安装最新版本。

  2. 创建 JavaScript 文件: 使用文本编辑器（比如 VSCode、Sublime Text、Notepad++ 等）创建一个 JavaScript 文件，并保存为 .js 扩展名。例如，你可以创建一个文件名为 example.js。

// example.js
console.log('Hello, World!');
1
2
  1. 在命令行中运行: 打开命令行工具（命令提示符或终端），进入到存放你的 JavaScript 文件的目录。然后运行以下命令：
node example.js
1

这将执行 example.js 文件中的 JavaScript 代码，并在命令行中显示输出，即 Hello, World!。

# 二、在 VSCode 中运行 JavaScript 文件

  1. 打开 VSCode，并打开你的 JavaScript 文件。

  2. 右键单击文件编辑区域的空白处，选择 "Open in Terminal"（在终端中打开）。

  3. 在终端中输入以下命令并按 Enter：

node filename.js
1

其中，filename.js 替换为你实际的 JavaScript 文件名。

以上步骤适用于大多数操作系统，包括 Windows、macOS 和 Linux。确保你已经正确安装 Node.js，并能够在命令行中运行 node 命令。
