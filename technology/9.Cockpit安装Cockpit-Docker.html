# A9. Cockpit安装Cockpit-Docker

新版的Cockpit不再提供Docker管理，改换为podman。针对新版Ubuntu可以用下列方法自己增加，Debian服务器也类似。

# 1. 下载插件

cockpit-docker.tar.gz (opens new window)

# 2. 拷贝或移动到Cockpit 目录

例如 /usr/share/cockpit，然后简单解压即可：

$ tar xf cockpit-docker.tar.gz -C .
1

重新启动Cockpit即可以发现Docker菜单。如果系统密钥安装Docker，要先安装，例如：

$ apt install docker.io
1

Enjoy it ~ ❤️
